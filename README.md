# Sloanes Knights
This is a simulator website for Sloane's Knights game, which is described in further detail in [this Numberphile video](https://www.youtube.com/watch?v=UiX4CFIiegM) as well as [this OEIS article](https://oeis.org/A392177).

The website is currently hosted [here](https://motiwala.ca/sloanes-knights).

## Deployment
This project uses `bun`. Run `bun run dev` to start the development server and go to `http://localhost:3000/sloanes-knights` in your browser.
